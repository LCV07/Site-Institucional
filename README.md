# Site Institucional — Empresa Exemplo

Este é um exemplo de site institucional com múltiplas seções, navegação intuitiva e layout responsivo.

Como usar:

1. Abra `index.html` no seu navegador ou use uma extensão como Live Server.
2. Os recursos estão em `styles.css` e `script.js`.
3. Substitua as imagens em `assets/images/` por arquivos reais.

Estrutura:

- `index.html` — página principal (single page)
- `styles.css` — estilos e responsividade
- `script.js` — comportamentos (menu móvel, smooth scroll, formulário demo)
- `assets/images/` — imagens de demonstração

Se quiser, posso:
- Gerar imagens de placeholder ou SVGs
- Adicionar páginas internas (sobre, serviços, contato)
- Preparar deploy (GitHub Pages) com instruções
